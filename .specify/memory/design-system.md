# Design System

---

## Color Palette

```
Background:   #0A0A0F   (deep dark)
Surface:      #13131A   (card background)
Surface2:     #1C1C28   (elevated card / popover)
Border:       rgba(255,255,255,0.08)
Text Primary: #F1F5F9
Text Muted:   #64748B
```

---

## Sport Colors (for calendar chips and badges)

```
NFL:    #D00000
NBA:    #1D428A
MLB:    #002D72
NHL:    #1F1F1F  (with white border)
Soccer: #00A650
NCAA:   #FF6B00
UFC:    #D4AF37
Golf:   #3A7D44
NASCAR: #FFD700
```

---

## Typography

- Headings: `Syne` (Google Fonts) — bold, modern sports feel
- Body: `Inter` (Google Fonts) — clean and readable
---

## Theme

- **Dark mode only** — no toggle needed
- Glassmorphism effect on cards: `backdrop-blur-sm bg-white/5`
- Glowing border on major event cards: `shadow-[0_0_20px_rgba(255,100,0,0.3)]`
